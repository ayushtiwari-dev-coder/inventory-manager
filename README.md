# Inventory Manager

CLI-based Inventory Management System built with Python and MySQL.

## Features
- Product management (add, update, delete, view)
- Sales tracking
- Stock validation
- Duplicate product protection
- Profit margin validation

## Project Structure

database/
- connection.py
- sql_handler.py

inventory/
- product_manager.py
- sales_manager.py

utils/
- validation.py

main.py
